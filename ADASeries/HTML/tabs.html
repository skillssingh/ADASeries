Why do we need Web Accessibility?

To make sure that your Web application is available to all irrespective of the disability a person can have. Assistive technologies such as screen readers play a big role in bridging the gap, but they only work well if your web site has:

  1. Correct use of HTML5 Semantics,
  2. CSS Harmony: it supports 4.05:1 color contrast,
  3. Proper use of ARAI Attributes &
  4. Good execution of Javascript to change the state of ARIA Attributes.
